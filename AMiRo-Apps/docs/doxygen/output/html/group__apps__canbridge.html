CAN Bridge initialization function.

Parameters
[in]canCan object to initialize. Must not be NULL.
[in]subscriberList with CANBridge subscriber information. Can be NULL if the CANBridge should not transmit topic data.
[in]publisher_topic_idsList with topic ids where the CANBridge should listen to. Can be NULL if the CANBridge should not receive topic data.
[in]num_publisherNumber of Publisher ids.
[in]servicesList with CANBridge service information. Can be NULL if the CANBridge should not transmit service data.
[in]request_service_idsList with all service IDs where the CANBridge should listen to. Can be NULL if the CANBridge should not receive service data.
[in]num_requestNumber of request ids.
[in]prioPriority of the execution thread.
