float amiro_dmc_config::factor

Factor to add new data points to the low-pass filtered error history, normalized to a time delta of one second. Must be positive.

With each iteration, the low-pass filter 'lpf' is updated with the latest error value 'e' wrt. the time delta between iterations 'dt' and this factor 'f' according to this formula: lpf = (max(f * dt, 1) * e) + ((1 - max(f * dt, 1)) * lpf)
