◆ URT_THREAD_MEMORY

#define URT_THREAD_MEMORY (   varname,
  stacksize 
)    THD_WORKING_AREA(varname, stacksize)

Macro function to layout a thread memory.

The overall memory required for a thread usually is a combination of the stack and some further data (i.e. the thread object itself). The layout in memory as well as the metadata may differ between kernels and event hardware ports.

Parameters
[in]varnameThe name of the resulting buffer variable.
[in]stacksizeThe size of the thread's stack.
