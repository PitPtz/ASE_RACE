aos_ssspmoduleid_t aos_ssspdata_t::modulecount

Number of SSSP modules.

A value of 0 indicates an uninitialize or invalid count. If moduleId is invalid, but this variable is greater than zero (MSI failed), the sccumulated number of modules is a lower bound.
