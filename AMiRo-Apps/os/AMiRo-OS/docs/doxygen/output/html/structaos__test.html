aos_testshellcallback_t aos_test::shellcb

Callback function to serve as wrapper between shell and test.

The purpose of this wrapper is to be able to execute tests programatically exactly like a shell command from the CLI. It should handle any required setup depending on given arguments and eventually execute the test.
