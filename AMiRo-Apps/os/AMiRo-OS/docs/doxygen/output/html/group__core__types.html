typedef uint8_t aos_status_t

Status type generally used in AMiRo-OS.

By definition, the most significant bit (MSB) indicates whether an error occurred (e.g. the function call failed). All other bits can be used to specify the issue in more detail. If the MSB is unset, all other bits can still be used to indicate and specify non-critical warnings. As a result, a value of 0 (no bit set) indicates a flawles success.
