Detailed Description

Core components of µRT.

Core components of µRT comprise configuration, primitive type definitions, core API, node API and the synchronization subsystem. All but the last of these components are central components of µRT and thus are always present. The synchronization subsystem is optional, though, and can be disabled via the configuration.
