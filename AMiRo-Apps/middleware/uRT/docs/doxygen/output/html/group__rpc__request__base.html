◆ urt_request_retrievepolicy

enum urt_request_retrievepolicy

Policies for retrieving previously submitted requests.

Enumerator
URT_REQUEST_RETRIEVE_LAZY 

Abort retrieval if a locked mutex or a bad owner is detected.

URT_REQUEST_RETRIEVE_DETERMINED 

Abort retrieval only if a bad owner is detected.

URT_REQUEST_RETRIEVE_ENFORCING 

Always retrieve, even if this cancels the request.

◆ urtRequestRetrieve()

urt_status_t urtRequestRetrieve ( urt_baserequest_t *  request,
urt_request_retrievepolicy_t  policy,
size_t *  bytes,
urt_delay_t *  latency 
)
inline

Retrieve a request.

Parameters
[in]requestThe request to retrieve. Must not be NULL.
[in]policyPolicy how to retrieve the request.
[out]bytesSize of the payload to be returned by reference (optional). May be NULL.
[out]latencyThe latency to be returned by reference (optional). May be NULL.
Returns
Status code indicating the result of the function call.
Return values
URT_STATUS_OKRequest was retrieved normally. The request is locked.
URT_STATUS_REQUEST_BADOWNERRequest had to be removed from a service. If policy was set to URT_REQUEST_RETRIEVE_ENFORCING, the request is locked.
URT_STATUS_REQUEST_LOCKEDRequest could not be acquired due to a locked mutex. Only occurs if the policy was set to URT_REQUEST_RETRIEVE_LAZY. Request is not locked.
URT_STATUS_DEADLINEVIOLATIONRequest was retrieved, but deadline has been violated. The request is locked. Only occurs for FRT and HRT requests.
URT_STATUS_JITTERVIOLATIONRequest was retrieved, but jitter has been jiolated. The request is locked. Only occurs for FRT and HRT requests.

Submit a request.

Parameters
[in]requestThe request to be used for this call. Must not be NULL. Must have been acquired before.
[in]serviceThe service to be called. Must not be NULL.
[in]bytesSize of the payload in bytes.
[in]maskEvent mask to be notified when the request is answered. Set this to 0 for "fire and forget" requests.
[in]deadlineDeadline offset until when this request is valid. A value of 0 indicates that there is no deadline. Only applies to FRT and HRT requests. Ignored if URT_CFG_RPC_QOS_DEADLINECHECKS false.
Returns
Always returns URT_STATUS_OK.
