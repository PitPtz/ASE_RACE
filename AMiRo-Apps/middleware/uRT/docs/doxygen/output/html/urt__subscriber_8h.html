◆ urt_hrtsubscriber_recovery_f

typedef void(* urt_hrtsubscriber_recovery_f) (void *params)

Recovery callback function type for HRT subscribers.

forward declarations

Recovery callbacks can be used to initiate recovery attempts in case some timing violation occurred. However, callbacks are only executed if the current value of the subscriber's violationCode member is URT_STATUS_OK. If a recovery has been initiated, it is the responsibility of the recovery logic to reset that variable to this value.
