Detailed Description

Topics mediate communication.

Topics act as mediators for publish-subscribe communication. Publishers pass their message to a topic, which is buffered there. Then, the topic informs all associated subscribers about the new message, but does not pass on the data. Instead, the subscribers have to fetch the data from the buffer ansynchronously.
