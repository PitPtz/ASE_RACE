◆ id

urt_messageid_t urt_message::id

Identifier to identify messages.

Messages identifiers are not globally unique but within a topic. The identifier is updated every time it is used to publish new data. A value of URT_MESSAGEID_INVALID indicates that the message has not been used yet.

◆ time

urt_osTime_t urt_message::time

Origin time of the information passed by the message.

This time represents the age of the information (e.g. sensor data). All real-time constraints are checked against this time.
