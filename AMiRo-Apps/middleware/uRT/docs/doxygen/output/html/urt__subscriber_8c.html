Fetch the latest message.

Parameters
[in]subscriberThe FRT subscriber that shall fetch the message. Must not be NULL. Must be associatied to a topic.
[out]payloadPointer where to copy the payload to. May be NULL.
[out]bytesPointer where to store payload size in bytes. May be NULL.
[out]timePointer where to store the information origin time to. May be NULL.
[out]latencyThe latency to be returned by reference (optional). Value is only modified if a message was fetched. May be NULL.
Returns
Status code indicating the result of the function call.
Return values
URT_STATUS_OKSuccessfully fetched a message.
URT_STATUS_FETCH_NOMESSAGEThere is no new message to fetch.
URT_STATUS_DEADLINEVIOLATIONLatency violated the maximum expected latency. Deadline violations are checked before jitter violations.
URT_STATUS_JITTERVIOLATIONLatency violated the maximum expected jitter. Jitter violations are only checked if no deadline violation was detected.
