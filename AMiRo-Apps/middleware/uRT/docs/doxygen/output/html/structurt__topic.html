◆ hrtReleased

urt_osCondvar_t urt_topic::hrtReleased

Condition variable to inform any waiting publishers that publishing is possible.

Publishing new messages is prevented if not all HRT subscribers have processed the oldest message in the list. Only when a message was processed by all HRT subscribers, it can be reused.

◆ hrtSubscribers

struct urt_hrtsubscriber* urt_topic::hrtSubscribers

Pointer to the first element in a list of associated HRT subscribers.

If URT_CFG_PUBSUB_QOS_RATECHECKS is true, HRT subscribers are ordered by their expected rate (most critical first). Otherwise, HRT subscribers are ordered by their deadline (most critical first). This pointer is NULL if there are no HRT subscribers associated to the topic.
